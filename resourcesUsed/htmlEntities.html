Html Entities

There are times when we have to use some elements that are common to the html docuement for example if i try to write open angular bracket in the html file the text editor will misinterpret it for a unvalid tag therefore inorder to use it we use Html Entities which is nothing but code snippets which indirectly tells the browser engine to generate the respective html element. Generally Html Entities looks like &entity_name; for example less than is written as (& nospace lt;) which gives us '<'.
Serial No. ELement Code Snippet Entity Number
1. Non-breaking space ( ) & nospace nbsp; & nospace #160;
2. Less than (<) & nospace lt; & nospace #60;
3. Greater than (>) & nospace gt; & nospace #62;
4. Ampersand (&) & nospace amp; & nospace #38;
5. Double quotation Mark (") & nospace quot; & nospace #34;
6. Single quotation mark (') & nospace apos; & nospace #39;
7. Cent (¢) & nospace cent; & nospace #162;
8. Pound (£) & nospace pound; & nospace #163;
9. Yen (¥) & nospace yen; & nospace #165;
10. Euro (€) & nospace euro; & nospace #8364;
11. Copyright (©) & nospace copy; & nospace #169;
12. Registered Trademark (®) & nospace reg; & nospace #174;
